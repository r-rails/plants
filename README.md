# README
Install ruby 3.1.2:

`rbenv install 3.1.2`

1. bundle
2. rails db:create
3. rails db:migrate

run `bin/dev`

gem 'administrate' is installed.
Administrate is a library for Rails apps that automatically generates admin dashboards.
Dashboard is available at: `/admin`

app scheme at: https://www.tldraw.com/r/1657013053860
